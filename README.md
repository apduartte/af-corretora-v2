# AF Corretora de Seguros e Saúde — Landing Page MVP

Landing page estática criada para captação de leads e contatos via WhatsApp.

## Objetivo

Gerar pedidos de cotação para seguros de saúde, odontológico, auto, residencial, vida, empresarial e consórcio.

## Estrutura

- index.html: estrutura da página e SEO
- assets/css/styles.css: design responsivo
- assets/js/app.js: interações e eventos
- assets/js/config.js: configurações de WhatsApp, analytics e endpoint
- assets/img/hero.svg: imagem principal
